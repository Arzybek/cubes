Игра "Кубики"
Версия 1.0.0
Автор: Казикулов Арзыбек (arzybek.kazikulov@gmail.com)
Описание:
Данное приложение является реализацией игры «Кубики» 
(http://www.funnygames.ru/igra/kubiki_1.html)
Требования:
Python версии не ниже 3.4
PyGame
Состав:
Графическая версия: cubes.py
Файл рекордов: saves.txt
Лог разработки: version.txt
Подробности реализации:
Поддерживается undo/redo.
game.py - родительский класс игры с основным циклом
cubes.py - реализованная игра, со всеми объектами, наследуется от game.py
cube.py - класс кубика, наследуется от GameObject
point.py - класс точки
config.py - конфигурационные данные
Game_object.py - родительский класс игрового объекта
tests.py - файл с тестами
text_object.py - класс текстового объекта
TextBox.py - класс ввода данных, наследуется от GameObject
button.py - класс кнопки, наследуется от GameObject
